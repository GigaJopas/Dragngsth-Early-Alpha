# Dragngsth EARLY ALPHA
Little guy that you can move around. Still working on it. Made with Pygame.

Circle button - clear clones

Button 1 - cloning on score increase

Button 2 - running away (from cursor) movement

Button 3 - running towards (cursor) movement

Button 4 - random movement (compatible with other move-types)

Slider - change movespeed

Button 5 - sprite spinning

Switch - switch skin

Buton 6 - sliding (compatible with other move-types)
